Display

block

Faz um elemento se comportar como um bloco,ocupando toda a linha

inline

Não quebra a linha,só ocupa o tamanho do elemento

inline block

Vai manter o posicionamento conforme o tamanho do conteudo,mas podemos usar width,heigth,margin e paddings

none

O elemento não aparece ma tala e não ocupa espaço

Tamanho e dimencionamento

  • width: Define a largura exata do elemento
  • max-width: Define a maxima largura do elemento
  • heigth: Define a altura exata do elemento
  • min-heigth: Define a altura minima que elemento pode ter

Margens(Espaçamento Externo)

  • margin-top: Define a margem superior do elemento
  • margin-rigth: Define a margem a direita do elemento
  • margin-bottom: Define a margem inferior do elemento
  • margin-top: Define a margem a esquerda do elemento
  • margin: Define todas as margem de uma vez
  • margin: Define todas as margem de uma vez

Padding(Espaçamento Interno)

  • padding-top: Define o espaçamento interno superior do elemento
  • padding-rigth: Define o espaçamento interno a direita do elemento
  • padding-bottom: Define o espaçamento interno inferior do elemento
  • padding-top: Define o espaçamento interno a esquerda do elemento
  • padding: Define todas as o espaçamento interno de todos os lados de uma vez

Bordas e Raio

border-radius

Define o raio da curva nos canto do elemento.Podemos usar em botões,imagens,tabelas...

Plano de Fundo

background-color

Define a cor de fundo de um elemento

background-image

Define a imagem de fundo de um elemento

Cores e Tipografia

color

Define a cor do texto,podemos definir de 3 forma

  • Hexadecimal(#ffffff #fff)
  • RGB(70,30,120)
  • Nome da cor(white,black,yellow...)

font-family

Define a fonte(Tipo de Fonte) usada no texto

font-size

Define o tamanho do fonte

line-height

Define a altura da linha
